## ❓ Troubleshooting

### "Freighter wallet not found"
- Make sure you installed the Freighter extension
- Refresh the page
- Try clicking "Connect with Freighter" again

### "Transaction failed: op_src_no_trust"
- You need to add USDC trustline first
- Go to Setup page and click "Add USDC Trustline"

### "Transaction failed: op_no_destination"
- The recipient address doesn't exist
- Use the test address I provided above

### "Insufficient balance"
- Make sure you got testnet XLM from Friendbot
- You need at least 1 XLM for transaction fees

### "Can't see my transaction"
- Click the "Refresh" button on the History page
- Wait 5-10 seconds for the network to process
